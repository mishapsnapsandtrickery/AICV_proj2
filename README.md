# YOLO와 VLM를 활용한 교량 외관결함 자동 점검 시스템

## 1. 프로젝트 개요
- 30년 이상 노후화된 교량은 2023년 현재 20%, 10년 후에는 59%까지 증가할 것으로 예상되어 유지 관리에 대한 필요성 증가
- 기존 점검 방식은 인력 중심의 점검 방식을 채택하며, 시간과 비용적 측면에서 비효율성을 띄고, 객관적 정량 평가의 어려움이 존재

## 2. 목표 및 기대효과
-  YOLO 기반의 세그멘테이션 모델을 활용하여 교량 외관 결함을 자동 탐지 및 분류
-  VLM과 연계하여 점검 결과를 자동 분석 및 요약하는 지능형 교량 외관 점검 시스템 개발
-  AI 비전 기술과 자연어 처리 기술의 유향을 통해 교량 유지관리의 디지털 전환 및 자동화를 실현하는 데 기여할 것으로 기대
